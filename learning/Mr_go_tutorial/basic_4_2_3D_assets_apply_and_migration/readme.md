## 이주를 활용하는 방법
### 나나이트로 에셋을 받으면 용량이 매우 크다. 만약에 협업을 하는 경우나 다른 프로젝트로 계속 옮겨야 하는 경우에는 용량 문제가 발생한다.
### 그래서 한 프로젝트에서 받고 싶은 에셋을 다운로드 받고, 거기서 받고 싶은 것만 다른 프로젝트로 이주한다. 그러면 용량 문제를 어느 정도 해결할 수 있다.
